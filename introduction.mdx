KYA Hukuk & Danışmanlık is a professional law firm website built with Flask, featuring a complete content management system and admin dashboard. The platform enables lawyers to manage their practice areas, team profiles, articles, and client communications.

## Overview

This Flask application provides a full-featured website for a law firm with both public-facing pages and a secure admin panel. The architecture supports flexible deployment on Vercel with PostgreSQL (Neon) or local development with SQLite.

## Architecture

The application is structured as a monolithic Flask app with the following components:

- **Flask 3.0.3** - Web framework with Jinja2 templating
- **SQLAlchemy** - ORM with support for PostgreSQL and SQLite
- **Flask-Login** - User session management with 3-hour timeout
- **Flask-WTF** - CSRF protection for forms
- **Werkzeug** - Password hashing and secure file handling
- **Pillow** - Image processing for uploads
- **Gunicorn** - Production WSGI server

## Database models

The application defines seven core models in `models.py:1`:

- **User** - Admin users with password hashing
- **SiteSetting** - Key-value store for site configuration
- **HeroSection** - Per-page hero banner content
- **TeamMember** - Lawyer profiles with ordering
- **PracticeArea** - Service offerings with icons
- **Article** - Blog posts with publish/draft workflow
- **ContactMessage** - Client inquiries with read tracking

## Public routes

The application provides these public pages defined in `app.py:251`:

- `/` - Homepage with featured practice areas and latest articles
- `/hakkimizda` - About page with firm information
- `/ekibimiz` - Team page with lawyer profiles
- `/faaliyet` - Practice areas page
- `/makaleler` - Articles/blog listing
- `/makaleler/<slug>` - Individual article view
